## 📚 Topics Covered

### 🗄️ SQL ✅
- SELECT, WHERE, ORDER BY, GROUP BY, HAVING
- Aggregate Functions — COUNT, SUM, AVG, MIN, MAX
- Joins — INNER, LEFT, RIGHT, FULL OUTER
- Subqueries & Nested Queries
- Window Functions — RANK, ROW_NUMBER, LEAD, LAG
- Practice Problems & Real-world Queries

### 🐍 Python ✅
- Core Basics — Variables, Loops, Functions, Data Structures
- List Comprehension & String Manipulation
- Exception Handling — `try` / `except` / `finally`
- File Handling & CSV
- Modules & Packages
- Object-Oriented Programming — Classes, Objects, Inheritance, Polymorphism, Encapsulation
- Iterators & Generators
- Sorting — Bubble, Selection, Insertion, Merge, Quick, Built-in `sorted()`
- Searching — Linear Search, Binary Search
- Time Complexity & Big-O Basics
- Debugging Basics

## 💡 Learning Approach

I write these notes keeping one thing in mind — **make it simple**.

- Concepts are explained in plain English first
- Code examples are short, readable, and well-commented
- Topics build on each other step by step
- No overwhelming theory — just what you actually need to know
